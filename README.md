# LSPP 2026 Assignment

A React.js and TypeScript implementation of the **Leapfrog Student Partnership Program (LSPP) 2026** website, reimagined with a modern, interactive user experience.

Rather than recreating the original page, this project focuses on enhancing usability, engagement, and visual appeal through animations, interactive components, and a responsive design.

## 🌐 Live Demo

- **Official Website:** https://studentpartner.lftechnology.com/
- **My Version:** https://lspp-react-website.vercel.app/

## Project Overview

The goal of this assignment was to redesign the LSPP website from a student's perspective while preserving its core purpose. The project explores how thoughtful UI/UX improvements can make information more engaging and easier to navigate.

### Highlights

- Interactive confetti effects
- Scroll-based animations
- Animated statistics section
- Searchable and filterable Student Partners directory
- Fully responsive layout
- Modern UI with improved visual hierarchy
- Fast performance powered by Vite

## Tech Stack

- React.js
- TypeScript
- Vite
- Tailwind CSS

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/preeyankakc037/lspp-react-website.git
```

### Navigate to the project

```bash
cd lspp-react-website
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
